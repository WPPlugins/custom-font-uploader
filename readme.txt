== Description ==

Custom Font Uploader is a powerful, extendable, google webfonts plugin that helps you to use google fonts in your theme .
It also allows you to upload your own custom font to your site.
The distinct feature of plugin only loads the fonts you select.This is good for your page loading times, as you want your site to load fast.
 

== Easy to use UI ==

We have build UI so that you can select any of google font is joyful.It also have option for fancy font preview testing!! you can easily add or remove of fonts with just one click.

== Installation ==

== Minimum Requirements ==

* WordPress 3.5 or greater
*PHP version 5.2.4 or greater
*MySQL version 5.0 or greater

== Manual Installation ==
You can download and install the plugin using built in WordPress plugin installer.

The manual installation method involves downloading the plugin  and uploading it to your webserver via your FTP application.

1. Download the plugin file to your computer and unzip it.
2. Upload the entire "custom-font-uploader" folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
